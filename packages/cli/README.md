# @presencing/cli

> Command-line tools for [Presencing](https://github.com/presencing/presencing).

## Status

🌱 **0.0.x — early preview / brand placeholder.**

Real commands (`presencing init`, `presencing register-stream`, etc.) ship with v0.1.

Star [the main repo](https://github.com/presencing/presencing) to be notified.

## Install

```bash
npm install -g @presencing/cli
presencing --help
```
